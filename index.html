Planuri de Antrenament

Focus pe Piept, Umeri, Triceps.

  • Împins la piept cu bară: 3 seturi x 6-10 repetări
  • Împins deasupra capului cu bară (Presă militară): 3 seturi x 8-12 repetări
  • Împins la piept înclinat cu gantere: 3 seturi x 10-15 repetări
  • Ridicări laterale cu gantere: 4 seturi x 12-15 repetări
  • Extensii la cablu (sfoară/bară): 3 seturi x 10-15 repetări
  • Extensii deasupra capului (ganteră/bară/cablu): 3 seturi x 12-15 repetări

Focus pe Spate, Biceps, Picioare.

  • Îndreptări (Convențional) sau Genuflexiuni cu bară pe spate: 3 seturi x 5-8 repetări
  • Tracțiuni (priză pronată) / Tracțiuni la helcometru: 3 seturi x 8-12 repetări
  • Ramat la cablu din șezut / Ramat cu bară din aplecat: 3 seturi x 10-15 repetări
  • Tracțiuni la față (Face Pull): 3 seturi x 15-20 repetări
  • Presa pentru picioare / Extensii pentru cvadriceps: 3 seturi x 12-15 repetări
  • Flexii femurale la aparat (culcat/șezut): 3 seturi x 12-15 repetări
  • Flexii cu bară (dreaptă/EZ) / Flexii cu gantere: 3 seturi x 10-15 repetări
